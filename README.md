# Rota Fluminense - Back-end

API do projeto Rota Fluminense, responsável pelo cadastro e consulta de locais turísticos do estado do Rio de Janeiro.

---

## Instalação

### 1. Clonar o repositório
```bash
git clone <url-do-repositorio>
cd rota_fluminense_backend
```

### 2. Criar e ativar ambiente virtual
```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 3. Instalar dependências
```bash
pip install -r requirements.txt
```

### 4. Executar a API
```bash
python app.py
```
