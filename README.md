# Rota Fluminense - Back-end

API do projeto Rota Fluminense, um sistema para cadastro e consulta de locais turísticos do estado do Rio de Janeiro

## Tecnologias
- Python
- Flask
- SQLite

## Como executar
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```
